#### 文件说明
```
src/tasks/ 任务类
src/config.py 项目配置
tests 自动化测试用例
deploy.txt 同步到更新库的文件列表, 如tests文件夹
main.py 入口
main_debug.py debug入口
pyappify.yml 打包exe配置文件
i18n 国际化文件, 可选
assets cv2使用的template, 需要使用coco格式
.github/workflows/build.yml 自动化构建任务
```
